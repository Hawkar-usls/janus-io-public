## What The Secret Is

The sidecar translates the operator's back/forth idea into a mechanical
observer pipeline:

```text
THOUGHT/GOAL -> target non-control accepted/rare-tail glyph surfaces
ATTENTION_FILTER -> open_sweep_score
EMOTION_STATE -> belief / resistance / gratitude weights
DECISION_BIAS -> forward / backward / mirror shadow motion
ACTION -> write intention only; JANUS scheduler is not modified
FEEDBACK -> accepted/z/repeat/job diversity reinforces the anchor
```

The important boundary is that this is written as evidence and shadow
intention. It does not alter nonce traversal, header bytes, submit policy,
extranonce, target, or the randomized traversal mirror.
