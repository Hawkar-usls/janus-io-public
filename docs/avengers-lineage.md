# Avengers Lineage

Status: explanatory note for public readers.

`Avengers` is a JANUS codename for bringing prior Proof-of-Work ideas into one
controlled benchmark frame. It is not a claim that any of the people below are
involved with JANUS, endorse JANUS, or are responsible for its results.

The point is respect:

```text
do not erase the founding work
do not pretend JANUS appeared from nowhere
cite the lineage, keep the claim narrow, add our own evidence
```

JANUS is a PoW scheduler benchmark. It does not claim a SHA-256 break. It asks
whether structured traversal can produce a measurably different accepted-share
rare-tail telemetry profile than a randomized traversal mirror under the same
frozen wire.

## Why "Avengers"

The name is a readable metaphor for a team of prior ideas:

- verifiable cost;
- hash-based work;
- peer-to-peer money;
- reusable proof discipline;
- structured mathematical work;
- memory-bound and graph-bound work;
- pool/job/wire discipline;
- reproducible benchmarking.

JANUS is the unruly teammate in that room. It uses the existing canon, preserves
the old evidence, and adds a controlled scheduler/corpus experiment on top.

This table is intentionally about ideas, not ownership. JANUS does not absorb
these works. It stands near them, cites them, and tests a narrower question.

## The Narrow JANUS Claim

JANUS should be described as:

```text
a controlled Proof-of-Work traversal benchmark that compares structured
scheduler behavior against a randomized traversal mirror under frozen
double-SHA256 wire rules.
```

JANUS should not be described as:

```text
a SHA-256 break
a guaranteed mining advantage
a nonce predictor
a replacement for Bitcoin, Primecoin, Cuckoo Cycle, or Equihash
```

## Avengers Rule

Avengers means:

```text
old work remains visible
new work remains testable
raw artifacts stay separate from generated reports
the randomized traversal mirror remains clean
the frozen wire remains frozen
```

That is how JANUS can be strange without becoming sloppy.
