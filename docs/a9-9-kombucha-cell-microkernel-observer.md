# A9.9 Kombucha Cell Microkernel Observer

A9.9 is a controlled strict 50/50 Proof-of-Work traversal benchmark. It
compares a JANUS structured traversal arm against a randomized traversal mirror
under the same frozen wire, same machine, same pool job stream, and same
desktop-load envelope.

The A9.9 addition is the Kombucha Cell Microkernel observer. It labels the
state of the traversal ecology without changing scheduling behavior.

## Purpose

The current question is intentionally narrow:

```text
Can JANUS structured traversal produce a measurably different accepted-share
rare-tail telemetry profile than a randomized traversal mirror under equal
exposure?
```

This is not a SHA-256 break claim, not nonce prediction, and not a block-finding
claim. The benchmark measures traversal behavior around an intact double
SHA-256 work function.

## What Changed

A9.9 adds observer-only Kombucha Cell Microkernel telemetry:

- cell mode;
- microkernel pressure;
- paradox score;
- charge;
- nuclei count;
- top cell labels;
- whether the observer would block aggression.

These labels are written to accounting and dashboard artifacts for later
review. They are not allowed to alter the frozen wire or the active scheduler.

## What Did Not Change

- Stratum wire path remains frozen.
- Header, nonce, extranonce, previous hash, and submit handling remain frozen.
- Allocator remains off.
- Strict 50/50 exposure remains active.
- BunnyHop, ReJump, WitchHunter, and Sovereign Triad remain observer/accounting
  surfaces in this run envelope.
- `scheduler_effect = none` for the Kombucha observer.
- `wire_change_required = false`.

## Reading The Snapshot

A9.9 is healthy and the frozen wire remains clean. Both arms have reached
`best_z = 34` under equal checked work. The JANUS-side signal is concentrated in
Memory ReJump, which produced the current JANUS `z34+` rare-tail event from a
small ReJump corpus. The randomized traversal mirror is denser in lower and
middle tails at this snapshot: `z30+`, `z32+`, and `z33+`.

This is a useful accepted-share corpus snapshot, not a final advantage claim.
The strongest honest statement is:

```text
A9.9 shows a clean equal-exposure rare-tail telemetry comparison where JANUS
matches the mirror at best_z=34, while the mirror currently leads in z30/z32/z33
density and JANUS shows a notable Memory ReJump z34 event.
```

Keep private unless separately scrubbed:

- local machine paths;
- live dashboards and event streams.

## Current Verdict

A9.9 is good enough to document as a current controlled benchmark snapshot. It
is not yet a victory claim. The clean public message is reproducibility,
observer discipline, equal exposure, frozen wire, and honest rare-tail
telemetry.
