# Experiments

This directory is for curated experiment notes, analysis summaries, and
publication bundles.

Raw local evidence belongs under `janus_io_o1_runs/` and should remain private
until a scrubbed proofpack is explicitly prepared.

Current curated area:

- `o1-01/` - baseline O1 experiment notes and generated summaries.
- `a9-v32-broad-mixture-random-control/` - methodology and curated summaries for
  the V32 broad-mixture runner with inline random-control accounting.
- `a9-11-active-triune-sovereign-gate-50-50/` - strict same-run 50/50 JANUS vs
  randomized traversal mirror evidence draft.
- `a9-9-kombucha-cell-microkernel-observer/` - current strict 50/50 JANUS vs
  randomized traversal mirror snapshot with observer-only Kombucha Cell
  Microkernel telemetry.
